# Project purpose

The purpose of this application is to maintain a list of books that have been read, as well as books planned for future reading.
Books can be added using their ISBN or by searching with keywords such as the book title or author name.

This application will use the Finna open API, which you can explore at: https://www.kiwi.fi/spaces/Finna/pages/51842373/Finnan+avoin+rajapinta

In the first phase, the core functionality will be implemented, including adding and searching for books. 
In the second phase, image-based search will be added — for example, using a photo taken with a phone.

The technologies used are React/TypeScript and ASP.NET Core. Terraform will also be used for provisioning the infrastructure on Azure.
This application is intended to serve as a learning project for new technologies and as a portfolio piece.
